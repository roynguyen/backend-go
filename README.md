# Build RESTful API in go using gorrila framework 
### Install: 
clone source code from github:
```
git clone https://github.com/roynguyen/backend-go
```
set path to source code in your desktop:
```
export GOPATH= "$HOME/your_path"
```
### run: 
```
go run main.go 
```
### server start on:
http://localhost:8000
